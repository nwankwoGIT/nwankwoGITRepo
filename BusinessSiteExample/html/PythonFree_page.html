Pipe Capacity

Compute the capacity of a pipe given its radius in inches and length in feet
  
              Cap  = 3.141592/(4*144)/5.6145 * (id * id) * len  
              where:  
              id  = pipe  internal  diameter 
              len = pipe  length 
              pi  = 3.1432 
              Cap = bbls/linear ft

Solution Details


              import math
              def pcal (id, len = 1):
                cap = math.pi*(id*id)/(4*144)/5.6145*len
                return cap

Pipe Annular Capacity

Compute the annular capacity of a pipe given its radius in inches and length in feet
  
                Cap  = 0.7854/808.5 * (od - id) ** 2 * len  
                where:  
                id  = pipe  internal  diameter 
                len = pipe  length 
                pi  = 3.1432 
                Cap = bbls/linear ft

Solution Details


                
                import math
                def pcal (id, od, len = 1):
                  cap = math.pi*(od-id)**2/(4*144)/5.6145*len
                  return cap
