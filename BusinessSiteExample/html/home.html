C/C++ Language

C/C++ is a very powerful programming language. I have been programming in C/C++ from the early days of C++ debut. The beauty of the language lies it's low level interface with the operating system. It is quite a difficult language to program in but the rewards are huge. C/C++ interfaces with many other languages like R, Node.js and Python. While it is very poor in graphic visualization, its computing power and speed can be exploited by interfaving it with R which has powerful visualization credentials.
